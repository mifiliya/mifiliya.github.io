  • Замена NPC в поселениях
    Автор: Неизвестен ▼ ЗАГРУЗИТЬ (v1.0, 7 KB) КАК УСТАНОВИТЬ МОД

    Мод заменяет NPC в поселениях на других,указанных в файле конфигурации (например добавляет в поселения старших сестер)

    После первого запуска создается конфиг по пути: "Mad Island\BepInEx\config\deadlayn.madisland.EnhanceSpawnNPC.cfg" В конфиге можно настраивать список замены в формате 15:90:20 (заменить 15id на 90id c 20% шансом)
    Пример: Replacements = 15:90:20, 11:17:100, ...

  • Добавление H-сцен из сюжета
    Автор: Неизвестен ▼ ЗАГРУЗИТЬ (v1.0, 24 KB) КАК УСТАНОВИТЬ МОД

    Мод добавляет H-сцены из сюжета в меню персонажа

    После первого запуска создается файл по пути: «Mad Island\BepInEx\config\deadlayn.madisland.UnlockSS.cfg». В нем задаются параметры анимаций для NPC в следующем формате:
    Структура записи: NpcID, SceneNumber: IndexList: IndexObj: AnimationName: Loop, ...
    Пример: 5, 1: 1: 17: A_idle: true, 1: 17: A_Contact_01: false;

    • NpcID: ID персонажа (например, 5 для определённого NPC).
    • SceneNumber: Номер сцены для этого NPC (например, 1 для первой сцены, 2 для второй).
    • IndexList: Первое число индекса списка анимаций (например, 1 в паре 1-17).
    • IndexObj: Второе число индекса списка анимаций (например, 17 в паре 1-17).
    • AnimationName: Название анимации (например, A_idle).
    • Loop: true (анимация зацикливается, ждёт клика мыши для продолжения) или false (анимация проигрывается один раз и переходит к следующей).

      • Сцены разделяются точкой с запятой (;). Анимации внутри сцены — запятыми (,).

    Где брать для конфига анимации? В меню игры нажимаете «F12», справа пролистайте вниз и увидите кнопки с индексами (1-15, 1-8 и т.д.), это и есть списки анимации, которые нужно добавлять. Когда вы нажимаете справа на эти индексы (1-15, 1-8), то слева появятся кнопки с названиями анимации (A_idle, ...), которые можно воспроизвести. Название у кнопок в неправильном регистре(я не виноват). Чтобы получить правильное название в нужном регистре, то придется смотреть в логи (BepInEx/LogOutput.log) или в консоль игры после нажатия на кнопки с индексами(1-17, 1-9).
